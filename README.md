## Project Overview
Customer churn is one of the most critical challenges faced by businesses today. This project focuses on analyzing customer data to identify churn patterns, retention trends, and key factors influencing customer behavior.
Using Power BI, this project transforms raw customer data into meaningful insights through interactive dashboards and visual analytics. The goal is to help organizations improve customer retention strategies and reduce churn rate.

## Objectives
Analyze total customer base and customer status
Identify active and churned customers
Calculate overall churn rate
Analyze churn distribution across different customer segments
Identify key factors contributing to customer churn
Build interactive dashboards using Power BI

## Tools & Technologies
Power BI Desktop
Data Cleaning & Transformation (Power Query)
Data Modeling
DAX (Data Analysis Expressions)

## 2️⃣ Churn Distribution Analysis

Customer Retention Trends
